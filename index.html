El problema con Google Sheets + HTML estático:

Para que un HTML pueda leer datos de Google Sheets automáticamente, la hoja debe ser pública (compartida como "cualquier persona con el enlace puede ver"). Si está privada, el navegador bloquea el acceso por seguridad (CORS).

Tienes dos opciones:

Opción A — Google Sheets Pública (más fácil, sin backend)

  1. En tu Google Sheet → Archivo → Compartir → Publicar en la web → elige el sheet → formato CSV
  2. Google genera una URL tipo https://docs.google.com/spreadsheets/d/ID/export?format=csv&gid=892513436
  3. El HTML la lee cada vez que se abre y se auto-refresca cada X minutos

Opción B — Google Apps Script (más segura, hoja privada)

  1. En tu Google Sheet → Extensiones → Apps Script
  2. Pegas un pequeño script que sirve los datos como JSON
  3. El HTML consulta esa URL
